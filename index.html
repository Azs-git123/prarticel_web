Pengenalan Linux

Linux adalah sistem operasi open-source berbasis kernel Unix yang sangat populer di kalangan developer, sysadmin, dan pengguna teknis. Linux digunakan secara luas di server, superkomputer, embedded systems, dan Android.

Catatan: Tutorial ini menggunakan distribusi Ubuntu. Beberapa perintah mungkin sedikit berbeda di distribusi lain.
Cek Versi Linux
bash
$ uname -a
# Output: Linux ubuntu 5.15.0 #1 SMP x86_64 GNU/Linux

$ lsb_release -a
# Cek detail distribusi Ubuntu
Kenapa Belajar Linux?

Linux digunakan di lebih dari 90% server web di dunia. Menguasai Linux membuka peluang karir di bidang DevOps, cloud computing, dan system administration.

Perhatian: Pastikan kamu memiliki akses ke terminal Linux atau bisa menggunakan WSL (Windows Subsystem for Linux) di Windows.
